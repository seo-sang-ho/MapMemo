# MapMemo

사용자가 지도를 기반으로 원하는 위치에 메모(제목/내용)를 등록하고 조회할 수 있는 지도 메모 웹 서비스입니다.  
카카오맵 → 네이버지도 기반으로 전환되었으며, JWT 기반 로그인/회원가입과 토큰 재발급 기능을 지원합니다.

---

## 🚀 주요 기능

### 🔑 인증/인가
- JWT 기반 로그인 / 회원가입
- Access Token + Refresh Token 재발급 기능
- 로그아웃 시 토큰 삭제 및 클라이언트 알림 처리

### 🗺️ 지도 기능
- 네이버 지도 기반
- 지도 클릭 시 해당 위치에 메모 추가
- 등록된 메모 조회
- 마커 클릭 시 해당 위치로 이동
- 마커 인포윈도우 표시

### 📝 메모 기능
- 메모 생성 (제목/내용 입력)
- 메모 삭제
- 메모-회원 연동
- 카테고리 기반 조회 기능

### ⚙️ 환경 구성
- 개발/배포 yml 분리
- QueryDSL 도입
- Spring Security 기반 인증 구조 구성

---

## 📦 기술 스택

### **Backend**
- Spring Boot
- Spring Security
- Spring Data JPA
- JWT
- MySQL
- QueryDSL
